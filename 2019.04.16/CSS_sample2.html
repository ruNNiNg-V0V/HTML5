텍스트를 꾸미는 CSS 종류

  • 문장의 정렬

    text-align : left / center / light

  • 글자의 색상

    color : (색상코드) / (색상이름)

    * 색상코드 : #○○○○○○ (○ = 0~9 / a~f 입력가능)

    색상 적용 예시

  • 글자의 크기

    font-size : px / em / rm / vw / vh

    폰트 사이즈 예시 : 기본값

    폰트 사이즈 예시 : 10px

    1. 폰트 사이즈 예시 : 2em

    폰트 사이즈 예시 : 5vw;

    폰트 사이즈 예시 : 5vh;
